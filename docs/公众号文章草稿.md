## 它是什么

它挂在你桌面角落。你每写一行代码、每跟 Claude Code 完成一个任务，它都看在眼里：涨经验、升级、从「摸鱼 😴」一路进化到「卷王 🔥」。卷到深夜会给你撒花 🎉，几天没写会"饿"得冒 💤 提醒你。

**而且它真的会动、会跳舞、会跟你说话。**

## 三个我自己最喜欢的点

**1. 它是"活"的，不是一张静图。**
会呼吸、会眨眼、踩着影子、时不时自己蹦一下。小舞精灵更绝——平时优雅站着，你点她、或者你一动代码，她就跳一小段舞，跳完歇下。有张有弛，像个真的小家伙。

**2. 它实时懂你在干嘛。**
通过 Claude Code 官方 hooks，你每提交一个 prompt、每完成一个任务、每调一次工具，桌宠**当场**反应、即时涨经验。一个人写代码，旁边有个小家伙陪着卷——这感觉挺上头的。

**3. 隐私守得很死。**
所有数据只读你本机的 git 和 `~/.claude` 用量，**全本地、绝不外传**。而且只统计"发生了多少次"（提交数、提问数、任务数），**从不读你的对话内容或代码**。开源的，这句话你能逐行验证。

## 怎么用上它（3 步）

技术栈特别简单——**原生 JavaScript + Electron**，clone 下来就能跑。需要 Node 18+。

```bash
git clone https://github.com/jnMetaCode/codepet
cd codepet
npm install
npm start
```

跑起来后，桌宠出现在屏幕**右下角**。剩下两步都在界面里点：

1. **选宠物**：右键托盘 →「设置」→「🐾 选择宠物」，9 只任选，点一下即换。
2. **开实时联动**（推荐）：设置 →「⚡ 实时联动 Claude Code」→ 开启。它会装一个**安全的 hook**（只记"发生了什么"，绝不读 prompt 内容），之后你在 Claude Code 里提问、完成任务，桌宠**即时反应、即时涨经验**。随时一键卸载，不影响 Claude Code 正常使用。

**还能传自己的图，做专属桌宠。** 设置里有个「✨ 自定义宠物」——传一张图就生成一只：单张图做静态宠物，**传个九宫格它自动切成 9 帧、变成会跳舞的那种**。把你家猫的照片、自己画的小人、甚至喜欢的角色拖进去，桌面上就多一只独一无二的它（图也全本地，不外传）。

> 不想跑源码、想要现成安装包？这版我们也在打包 .dmg / .exe，**进群第一时间发**。

升级机制也很直白：经验 = 提交数×10 + 增删行数×0.1 + 提问数×4 + 完成任务数×8 + 工具调用×1，每 100 经验升 1 级；能量随时间衰减（约 3 天满→空），不写就会"饿"。全程本地 JSON，重启不丢。
