⚡ 实时联动 Claude Code（推荐）

装一个安全的 hook：你在 Claude Code 里提问 / 完成任务时，桌宠当场反应、即时涨经验。
hook 只记录"发生了什么"（事件名/工具名），绝不读 prompt 内容；随时可一键卸载，不影响 Claude Code 正常使用。

🐾 选择宠物

选一只你喜欢的。立绘未生成时显示 emoji 占位，生成后自动替换。（仅立绘风格）

✨ 自定义宠物（上传你自己的）

传一张图就能做一只专属桌宠。
· 单张图片 → 一只静态宠物（建议透明背景 PNG，正方形最佳）。
· 九宫格图片 → 自动切成 3×3 共 9 帧，做一只会跳舞的宠物。
